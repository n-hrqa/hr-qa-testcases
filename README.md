# HR QA テストケース（ポートフォリオ）

## 概要
HR系SaaS（人事労務システム）を想定し、
有給休暇・勤怠管理に関するテストケースを作成しました。
QA未経験ですが、実務を想定して「正常系・異常系・境界値」を網羅しています。

## 想定システム
- 人事労務SaaS（勤怠・有給管理）
- Webアプリケーション
- 従業員は日次打刻を行う

## 前提条件（例）
- 所定労働時間：1日8時間／週40時間
- 有給付与：入社6ヶ月後に10日、その後1年ごとに付与
- 有給取得単位：1日／半日／時間単位
- 深夜時間帯：22:00〜5:00
- 勤怠締め日：月末

## 作成内容
- 有給休暇のテストケース（付与／取得／時間休／繰越／異常系）
- 勤怠管理のテストケース（残業／深夜／休憩／締め処理）

## 工夫点
- 労働基準法の基本ルールを前提に期待結果を設計
- 残日数0、端数（0.5日／4時間）など境界値を意識
- 実務で発生しやすい異常系を重点的に作成
